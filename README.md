# Mars Rover Image Viewer

## Requirements

- Node.js (v14 or higher)
- npm (v7 or higher)
- Next.js
- TypeScript

## Getting Started

```bash
git clone <repo name>
cd mars-rover-image-viewer
npm install
npm run dev
npm run test (optional : for running tests)
```

## Environment Variables

Create an .env.local file in your Root directory

- API_KEY
- BASE_URL
